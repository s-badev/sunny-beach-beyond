# Sunny Beach & Beyond

Sunny Beach & Beyond is a modern interactive coastal guide for Sunny Beach, Nessebar, Sveti Vlas and Elenite. The project combines beach zones, nightlife, local notes, practical routes, a map-style place explorer and an archive-inspired Then & Now section, shaped by more than 20 years of visiting the area.

## Purpose

This project is a premium editorial coastal guide, personal local field guide, light interactive map explorer and archive-style storytelling project for Bulgaria's southern Black Sea coast.

It is designed to feel like a modern seaside travel guide, digital travel magazine and curated local archive, not a booking funnel or generic tourism template.

## Current Status

Early MVP / portfolio project.

The first version is a single-page portfolio MVP using local TypeScript data files and no backend.

## What This Project Is Not

This project is not:

- a hotel booking website
- a restaurant directory
- a real estate portal
- an admin dashboard
- a recipe or food app
- an authentication, payment or booking platform

It intentionally does not include a backend, database, auth, booking, payments or admin panel.

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
